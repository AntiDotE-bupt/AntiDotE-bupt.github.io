1 C语言语法基础

level1 常用语法 及 语法技巧 （约20小时）200题

头文件

头文件 #include <cstdio> 包含了 printf 和 scanf 函数
头文件 #include <iostream> 包含了cin 和 cout 函数
万能头 #include<bits/stdc++.h>

命名空间

使用命名空间 using namespace std，使得不同C++文件中的变量不会重名冲突

变量

变量类型

变量声明

输入输出

或者

输出多个变量

cout << a + b << ‘ ‘ << a * b << endl;

判断浮点数相等

fabs(sqrt(3) * sqrt(3) - 3 ) < eps

消除浮点数精度误差

round(a * 100) / 100.0

数组

初始化： int a[3] = {0, 1, 3}'
也可以不写长度： int b[] = {0, 1, 3}'
全部初始化：int f[10] = {0};
边长数组： vector<long long> pre(n + 1, 0); 长度为n+ 1，初始化为0
清空数组：memset(a, 0, sizeof a); 数组名、初始化成什么数字、字节数B
复制数组：memcpy(b, a, sizeof a); 把a复制给b，第三个参数为长度

字符数组

ASCII码表记忆  
48 0  
65 A  
97 a  

一种初始化方式： char a4[6] = "daniel";  
读入char s[]： scanf("%s", s);  
读入string: getline(cin, s);  
过滤回车： scanf("\n%c", &c);

字符串常用函数：  
strlen(s); 返回长度  
strcmp(a, b); 比较字典序大小 a - b，返回-1 0 1  
strcpy(s2, s1); 把s1赋值给s2;

字符串

标准库类型string, 包含在#include <string>
初始化： 

getline(cin, s); //包含空格的一次读入一行  

cin >> s; //不包含空格的读入一个字符串  

string s2 = s1; //支持赋值  

string支持<、>、==的比较，支持两个string用+号拼接； 

如何处理多组字符串输入：  while(cin >> s >> sub)  或者 while(getline(cin, A))  

字面值和string相加会转化为string，两端至少有一个stirng类型；

常用函数： 

s1.empty(); //返回0或1;  

s1.size(); //返回无符号整数，O(1);   

s.substr(idx, len);//返回s从idx为下标，len为长度的子字符串  

s.insert(idx + 1, sub); // 在原串下标为idx的字符串str前插入字符串s  

c = tolower(c)    c = toupper(c)// 转换大小写;  

s1.find(s2); // 在 s1 中查找字符串 s2，找到返回 s2 首字母在字符串中的下标，找不到返回 -1  
s1.rfind(s2); //反向查找  

s1.replace(pos, len, s2);  // 把 s1 中从下标 pos 开始的长度为 len 的子串替换为 s2    

s1.erase(it);  // 把 s1 字符串中迭代器 it 处的字符删除  

s1.erase(pos, len);// 把 s1 中从下标 pos 开始的长度为 len 的子串删除

函数

静态变量

static int cnt = 0` 初始化仅会在第一次调用时进行
相当于在函数内部开了一个只有这个函数能调用的全局变量，开在堆区避免爆栈以及避免变量重名

传参

传值参数
int max(int x, int y)
传引用参数
int max(int &x, int &y)
数组形参（多维数组只有第一位宽度可以省略）
int output(int n, int a[5]) 或 int output(int n, int a[]) 或 int output(int n, int *a)
默认参数void foo(int a, int b = 10)

类

类可以将变量、数组和函数打包在一起 

class Person  
{
    private:  
        int age, height;
        double money;
        string books[100];

    public:  
        string name;
        void say(){

        }
        int getAge(){
            return 0;
        }
};

结构体

结构体默认变量是public，类默认变量private。
二者都可以像变量一样声明
构造函数： 

struct Person 
{
    int age, height;
    double money;
    Person() {} //默认构造函数
    Person(int _age, int _height, _double) //带参数 构造函数
    {
        age = _age;
        height = _height;
        money = _money;
    }
};

指针和引用

  1. 指针:
    int* p = &a;， 定义一个指针p，指向a的地址
    输出地址(void*)&a

指针指向存放变量的值的地址，因此我们可以通过指针来修改变量的值，如： 

int a = 10;
int* p = &a;
*p += 5  
cout << *p;//读取
*p = 12;//赋值

  1. 引用，别名:
    int& p = a; 定义一个变量p，和a在同一个地址上面

  2. 链表：

    struct Node
    {
    int val;
    Node* next;
    Node(int _val) : val(_val), next(NULL) {}
    };

    Node* p = new Node(1);

如果用Node* p = new Node(1); 来创建链表节点，则用p->next
如果用Node p = new Node(1); 来创建链表节点，则用p.next

STL

  1. vector 变长数组
    头文件#include <vector> C++ 标准（特别是 C++11 及以上）并不支持变长数组，建议换成 vector

声明: 

vector<int> a({1, 2, 3});  
vector<int> b[233];  
vector<vector<int>> dp(n + 1, vector<int>(m + 1, 0));//二维数组

常用函数 : 

a.size();返回长度  
a.empty();返回true或false  
a.clear();把当前数组清空  
a.push_back();在最后加一个元素  
a.pop_back();从最后移除一个元素
  1. queue 队列
    #include <queue> 头文件，包含queue和priority_queue

声明： 

queue<int> q;  
priority_queue<int> q;大根堆  
priority_queue<int, vector<int>, greater<int>> q; 小根堆  
priority_queue<pair<int, int>> q;

常用函数: 

q.push();队尾插入  
q.pop();队头弹出  
q.front();返回队头   
q.back();返回队尾  
pq.push();插入元素  
pq.pop();删除堆顶  
pq.top();返回堆顶

  1. stack 栈

    头文件#include <stack>包含栈
    s.push(); 压栈
    s.pop();弹出栈顶

  2. deque 双端队列

常用函数: 

dq.front();取得头元素  
dq.back();取得尾元素  
dq.push_back();尾插  
dq.push_front();头插  
dq.pop_back();移除尾元素  
dq.pop_front();移除头元素  
dq.clear();清空队列
  1. set 集合

声明： 

set<int> s;元素不能重复  
multiset<double> s;可以包含若干个相等的元素

常用函数: 

s.empty();  
s.clear();  
s.size();  
s.insert(x);  
s.find(x);返回迭代器，不存在返回s.end()  
s.lower_bound(x);查找大于x的最小  
s.upper_bound(x);查找小于x的最大  
s.erase(x);删除所有x的元素，如果是迭代器就把迭代器删掉  
s.count(x);返回x的个数
  1. map 哈希映射
    声明：
    map<key type, value type> name;使用红黑树，查找插入O(logn)
    unoredered_map<int, int> umap;使用哈希桶，查找插入都是O(1)
    常用函数：
    m.insert({“a”, {}});
    m.find(x) != m.end();
    可以像访问数组一样插入查找m[i];

常用库函数

#include <algorithm>

  1. reverse 翻转
    翻转一个vector: reverse(a.begin(), a.end());
    翻转一个数组: reverse(a, a + n);

  2. unique 去重
    int m = unique(a.begin(), a.end()) - a.begin();
    int m = unique(a, a + n) - a;

  3. random_shuffle() 洗牌
    srand(time(0));
    random_shuffle(a.begin, a.end());

  4. sort() 排序
    sort(a.bgein(), a.end()); 从小到大
    sort(a.bgein(), a.end(), greater); 从大到小
    传入比较函数:
    sort(a, a + 5, cmp); 

     bool cmp(rec a, recb) {  
     return a.x > b.x;  
 }
