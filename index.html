anti的算法笔记

备战复试机试，本文学习内容来自Acwing语法基础班以及算法基础班，仅做个人笔记

1 C语言语法基础

level1 常用语法 及 语法技巧 （约20小时）200题

头文件

头文件 #include <cstdio> 包含了 printf 和 scanf 函数
头文件 #include <iostream> 包含了cin 和 cout 函数

命名空间

使用命名空间 using namespace std，使得不同C++文件中的变量不会重名冲突

变量

变量类型
bool 布尔类型 1表示true 0表示false
long long int 64位整形
long double 更长的浮点数

变量声明
long long l = 1000000000LL 最后的LL表示强制长整形，不加的话表示整数
float f = 12.34f 同理最后的f表示强制浮点数

输入输出

cin >> a >> b;
cout << a + b << endl;

或者
scanf(“%d %d”, &a, &b);
printf(“%d %d\n”, a + b, a - b);
保留两位小数的float浮点数 %.2f
double用%lf
long long用%lld

输出多个变量

cout << a + b << ‘ ‘ << a * b << endl;

判断浮点数相等

fabs(sqrt(3) * sqrt(3) - 3 ) < eps

消除浮点数精度误差

round(a * 100) / 100.0

数组

初始化： int a[3] = {0, 1, 3}'
也可以不写长度： int b[] = {0, 1, 3}'
全部初始化：int f[10] = {0};
清空数组：memset(a, 0, sizeof a); 数组名、初始化成什么数字、字节数B
复制数组：memcpy(b, a, sizeof a); 把a复制给b，第三个参数为长度

字符串

Markdown用法简单记录

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档

1.标题

用几个“#”号来表示几级标题

2. 段落

段落的换行是使用两个以上空格加上回车

3.字体

可以用 * 号或者 下换线_ 加在字体的两侧
一个表示斜体；两个表示粗体；三个表示粗斜体
如下：

5.删除线

在文字的两端加上两个~~即可，如：

BAIDU.COM

6.创建注脚

7.列表

Markdown 支持有序列表和无序列表
无序列表在星号* 加号+ 减号- 后面加上一个空格，在填写内容：

  • 第一项
  • 第二项
  • 第三项

有序列表使用数字并加上 . 号来表示，如：

  1. 第一项
  2. 第二项
  3. 第三项

列表嵌套
列表嵌套只需在子列表中的选项前面添加两个或四个空格即可：

  1. 第一项：
    • 第一项嵌套的第一个元素
    • 第一项嵌套的第二个元素
  2. 第二项：
    • 第二项嵌套的第一个元素
    • 第二项嵌套的第二个元素

区块引用是在段落开头使用 > 符号 ，然后后面紧跟一个空格符号：
